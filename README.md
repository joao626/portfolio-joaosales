Portfólio pessoal de **João Sales**, Desenvolvedor de Software. Projeto desenvolvido com HTML, CSS e JavaScript puros, seguindo boas práticas de semântica, responsividade e performance.

## Status

🚀 **Em produção** - [Acessar portfólio](https://joao626.github.io/portfolio-joaosales)

## Tecnologias Utilizadas

| Tecnologia | Descrição |
|------------|-----------|
| HTML5 | Estrutura semântica com `<header>`, `<nav>`, `<main>`, `<section>`, `<footer>` |
| CSS3 | Flexbox, Grid, Custom Properties, @keyframes, Media Queries |
| JavaScript | DOM manipulation, Intersection Observer, LocalStorage |
| Devicon | Ícones das habilidades técnicas |
| Font Awesome | Ícones das redes sociais |
| Google Fonts | Fonte Inter |

## Funcionalidades

- **Dark/Light Mode** - Toggle manual com persistência em localStorage
- **Design Responsivo** - Adaptado de 320px a 1920px
- **Animações Nativas** - Fade-in ao scroll, hover effects, transições suaves
- **Menu Mobile** - Navegação adaptada para dispositivos móveis
- **Navbar Dinâmica** - Oculta ao scroll para baixo, exibe ao scroll para cima

## Seções

1. **Hero** - Apresentação com foto, nome e CTA
2. **Sobre Mim** - Bio e trajetória profissional
3. **Habilidades** - Grid com 15 skills técnicas
4. **Projetos** - 3 projetos com links para demo e repositório
5. **Contato** - Redes sociais e email

## Como Rodar Localmente

1. Clone o repositório:
   ```bash
   git clone https://github.com/joao626/portfolio-joaosales.git
   ```

2. Abra o arquivo `index.html` no navegador:
   ```bash
   cd portfolio-joaosales
   start index.html
   ```
